# Thoughtful Skills

Skills for AI coding agents to use [Thoughtful](https://www.tokatsu.ai) as a persistent knowledge layer.

## Installation

```bash
npx add-skill Lex-Inc/thoughtful-skills
```

## Available Skills

### thoughtful

Persistent knowledge management using the Thoughtful CLI. Enables AI agents to:

- Store and retrieve knowledge across sessions
- Manage projects, ideas, and insights
- Capture higher-level strategy and product direction
- Set up CLAUDE.md/AGENTS.md for agent memory loops

## Requirements

- [Thoughtful CLI](https://www.npmjs.com/package/thoughtful): `npm install -g thoughtful`
- A Thoughtful account: [Sign up](https://www.tokatsu.ai/signup)
